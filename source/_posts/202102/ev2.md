#### TSLA 特斯拉

我完全看不懂特斯拉，过去看不懂，现在依然看不懂。

我第一次试驾 Model 3 是2019年6月8号，几乎无人能敌的做工和无比垃圾的刹车脚感就让我劝退，那时候最基础最入门的 Model 3 应该还需要40万左右，而一个看得过眼的后驱长续航版就需要40大了。

那时候我真的是看不懂这个车，从产品本身来说，对 Model 3 的理解就是这不就是一台电动福克斯吗，除了 Tesla 这个标 和 Musk 这个人对于品牌略有加成以外。在我理解中，40+ 万买入 Model 3 的人似乎只能是因为自己有这个预算而没有别的竞品才会购买。

后来，大家也都知道了，伴随一茬一茬韭菜掏出了自己的钱包，现在最入门的 Model 3 只需要24.99 万。还算是个说得过去的产品，也向着福克斯的价格考虑了，起码是个福克斯ST了。

我一直是TSLA的看空者，从上海TSLA建厂后，我看到国产 Model 3 的做工和小问题之后，我当时就想：这么糙的车，如果是那些贷款买车的刚需，他们与花40-50万买车人对于车的理解可能截然不同。“添大件”的思维方式也许会让他们对自己买的“豪车”容忍度比后者的容忍度低得多，于是我预计也许会有大量售后事宜拖垮TSLA，或者至少造成一个“现象级”的危机。

但，显然TSLA的高层和智囊能够很好拿捏这个问题，售后问题也不是没有，自燃、失控、衰减等也不是没有，只是都完全在控制之下。同时伴随产量和交付量，TSLA股价一路飙升一骑绝尘。

一方面，认知之外的钱不碰，所以自始始终我也没有买过TSLA的股票，也没有真正做空过TSLA。而另一方面，所谓敬畏市场指的就是，TSLA市值一路飙升到现在，就是一种正确和合理，我并不需要打破砂锅问到底地去给自己找一个能够说服自己的原因。

而，不得不承认，这个我看不懂的东西，构成了整个新能源行业的坐标系，形成了其他各家公司的参考。



#### 小鹏

小鹏这里，似乎有些分裂，甚至有些理想和现实的距离。

在虚拟世界里，有这样两个facts:
1.大家聊小鹏都是“轿跑”，甚至小鹏官网官图，P7在中间的竟然是剪刀门版本的轿跑车。仿佛对标的就是 Model S, in half price.

2.大家聊起小鹏，都用小鹏的NGP对比特斯拉的FSD，仿佛不买NGP就没有灵魂。

但，这两个 facts 在我的实地调研下，却是另一种倾向：
1.试驾车主推两驱中配长续航，NEDC670那个，高配看不到，性能版也看不到，剪刀门也看不到。

2.试驾车并未选配NGP，销售小哥从头到尾也没有给你推荐NGP，只是说我们辅助驾驶就够了，跟特斯拉也差不多，比ACC好用些仅此而已了。


那就意味着，虚拟世界中，小鹏是和特斯拉竞争运动、性能、配置等的产品，而现实世界中（至少北京金港店），销售口中描绘的和试驾车所塑造的，皆是一副我们是一个很好用的家用车的感觉。反正NGP估计你也不买，P版你也不买。

facts 之后的结论就是：

NGP 和 性能版，在产品序列定位上，根本就不是为了消费者的定位，而是为了投资者的定位。不负责瞎逼乱猜翻译一下就是：“我们小鹏也知道出高性能版和NGP也没有什么人买，但不出的话我们在投资人看起来就和TSLA差距太大了。”

于是，为了维持和TSLA的距离差，哪怕是为了作为资本市场眼中的公司竞争力和产品力，为了能够蹭到TSLA的保障，我们也需要让一个家用车带有各种TSLA才看中的额外属性。

但谁又能说TSLA不是为了维持高想象空间而搞这些玩意呢？



#### 理想

理想应该是这几个新能源赛道上最老实的那一个。从如下几个角度理解“老实”的含义：

1.产品
毕竟是[1.能够加油的]并且[2.只用电力驱动]的汽车产品。跟PHEV的区别在于发动机不参与驱动。有人说这是脱裤子放屁，但确实从产品划分上看这个 增程式 确实比 PHEV 更能给人靠近 BEV 的感觉，但和 BEV 比起来似乎也还差了半个台阶。技术上，看起来像是介于汉兰达和ES8中间的那个产品。

2.人群定位（对比蔚来）
上篇文章说，理想ONE基本上是精装修汉兰达。所以这些车的购买者必然是奶爸群体。我要说最惨的人就是30-45岁的男人，应该没有人反对吧。责任重、工作累、养家糊口、赚钱交给老婆、周末陪孩子，这些人有个屁的想象空间（此处对应蔚来）。

3.不选配
都是32.8万，顶多baby blue +1万，那这产品就是给人货真价实超大杯的感觉，for customers, good news, for investors, bad news. 小米2999好产品，用了好几年才把上市破发的股价追回来。

所以，可以说理想努力打造一个奶爸好产品（甚至我本人都有点心动，但没指标也没辙），但资本市场这边就是另一番景象了。


#### 蔚来

蔚来如同xie教一样，给关注着、车主、喷子、资本市场不断释放我们是一群不一样的人、我们是精英、我们是社会责任、我们是未来生活方式、我们是和BBA不一样的奢华，我们是不油腻的有钱人。

总之乱七八糟一大堆，从 NIO HOUSE 到 NIO DAY，从他们早年的EVE/EP9到它们请SDL 做的 NIO Formula E Branding，蔚来所有的努力只有一小部分在汽车，或者反过来说，蔚来在汽车之外还做了大量的努力。看你怎么想。

Which means, 别把我们蔚来当一个汽车公司，我们是生活方式公司，我们是社会精英圈子，我们是社会责任的代表，我们是 new money 的方向，我们是不油腻的中年。

Which means, 当你试驾的时候，你体会到的是那些不循规蹈矩的中年人生活，是一种价值和认可，是可以对着Q5L 或 Q7 喊出一声“油腻中年人”的狂妄。

Which means, 当这个社群组织起来，可以形成一种精英的阶层，一种如同下五子棋拿到双活三一般的满满的可能性。说的远一点，万一形成一个新晋中国精英阶层的代表也不是不可能；说的离谱一点，别忘了当年有个姓李的练气功的人就发现了一条组织大众的路线。

我上面这些瞎扯般的联想虽然离谱，但是基本想说的是在资本市场，你蔚来不是个老实人，卖车交个朋友，当你蔚来交了全中国大量精英阶层并能够引导或者引领些什么的时候，你说蔚来值多少钱？

据说，在上海金融圈，一辆蔚来已经是基本的社交门槛了。

#### 结论

所以，**简单总结**一下：

特斯拉TSLA：是个好公司，虽然看不懂，但是市场给了我们这种看不懂的人一个简单直接的答案。很难想象一家造车公司享受着比互联网公司还高得多的市盈率（哪怕按2021预测下的数据）。以及很难想象一家公司用不存在的期货FSD形成了一个公司在资本市场的丰厚溢价，且不说FSD好不好用，就玩儿FSD在资本市场溢价这一手，TSLA确实是屌。

小鹏：最接近TSLA的公司，以手踏踏实实做经济适用车，一手把自己包装成TSLA最大的竞争者。逻辑是：你们看TSLA市值7000多亿，我们小鹏做的事情和TSLA这么重合，总得值1/10个TSLA吧？ 什么？ 1/10 都够不着，1/20总差不多了吧。

理想：一个老实人公司，最接近传统汽车公司的公司，从资本市场反馈来看也是这样。用一种爆款产品的逻辑套入丰田的估值体系，再加上一点点BEV行业的溢价足矣。

蔚来：做车只是第一步，哪天看到 NIO 在商场里开个服装店、日杂店甚至咖啡店也完全不用意外。一方面用TSLA做个对标，再加入上面说的“精英群体可能性”的溢价，就是对蔚来的判断了。这也确实是蔚来可怕的地方。

#### 总结

所以，各家公司用一句话**大力总结**一下：

特斯拉：一直看不懂，超出我的认知，但市场说了算。我态度上持续看空但行动上不会操作。

理想：老实人一个，没什么太大想象空间，但可以期待后续爆款（据说是奶爸+）
蔚来：基本面最好，想象空间最大（但我猜市场早就兑现了这部分想象力）
