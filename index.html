开篇说明

之前一直在一个 self-host 的WordPress 上更新blog，然鹅慢慢的我发现self-host blog 惰性真的强。后来工作中接触了 git+atom 的协作方式，这样的协作确实很舒适。于是在20201223今天更新了 self-host blog 的后台编辑器，使其适应markdown，同时按教程安装了 Hexo 作为一个完全开源的基于 Markdown 的书写blog.

后续的更新路径是：
  1. Atom 中用 markdown 书写，同步至 github 并生成 Hexo 博客.
  2. 复制文字源码到 WordPress 网站后台的 markdow n编辑器手动同步
还没想好的问题
  1. 图片的更新：
    可能性1 用self-host 的VPS空间
    可能性2 部署阿里云OSS或者七牛云作为图床
  2. 如何同步之前已有的 WordPress 文字至 Hexo (也许并不会折腾了)
